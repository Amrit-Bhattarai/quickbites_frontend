- Agents/

- Users/

- Menu/

  - FoodList.jsx

  - ManageVariants.jsx
- Orders/
  - OrderList.jsx
  - AssignAgent.jsx
- Review/
  - MenuReview.jsx
- Earnings/

- Salaries/
  - AgentSalaries.jsx
- Notifications/
  - NotificationBell.jsx
- AdminDashboardLayout.jsx
- AdminDashboardRoutes.jsx

Each subfolder contains components related to a specific admin feature, making the codebase modular and maintainable.
